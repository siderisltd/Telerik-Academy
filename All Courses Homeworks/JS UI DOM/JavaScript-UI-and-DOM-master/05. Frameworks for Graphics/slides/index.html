Drawing Shapes with KineticJS

  • KineticJS has all the default shapes from Canvas, and some more:
Rectangular Circle 
rect = new Kinetic.Rect({
  fill: 'yellowgreen',
  stroke: '#CCCCCC', 
  x: 250,
  y: 350,
  width: 57,
  height: 93
});
 
circle = new Kinetic.Circle({
  radius: 45,
  fill: 'purple',
  stroke: 'blue',
  strokeWidth: 3,
  x: 450,
  y: 350,
});

Drawing Shapes with KineticJS

  • KineticJS has all the default shapes from Canvas, and some more:
Straight line Curved line 
straight = new Kinetic.Line({
  points: [x1, y1, x2, y2],
  stroke: 'green',
  strokeWidth: 2,
  lineJoin: 'round'
});
 
curved = new Kinetic.Line({
  points: [x1, y1, x2, y2],   
  stroke: 'green',
  strokeWidth: 2,
  tension: 1
});
